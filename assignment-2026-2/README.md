# Rendezvous Problem (Assignment 2026-2)

Σε αυτή την εργασία υλοποιείται ένα πρόγραμμα σε Python που λύνει το πρόβλημα συνάντησης (rendezvous) δύο ατόμων, 
της Alice και του Bob, πάνω σε έναν γράφο.

Οι κόμβοι του γράφου αντιπροσωπεύουν θέσεις και οι ακμές τις δυνατές μετακινήσεις. 
Η Alice και ο Bob ξεκινούν από διαφορετικούς κόμβους και κινούνται ταυτόχρονα σε κάθε χρονικό βήμα.

## Στόχος

Να βρεθεί αν είναι δυνατόν να συναντηθούν στον ίδιο κόμβο την ίδια χρονική στιγμή.

Αν δεν είναι δυνατόν:
- Στον μη κατευθυνόμενο γράφο προστίθεται μία ακμή.
- Στον κατευθυνόμενο γράφο προστίθενται μία ή δύο ακμές ώστε να δημιουργηθούν κύκλοι.

## Περιγραφή λύσης

Η λύση βασίζεται σε BFS (Breadth-First Search).

Αρχικά γίνεται ανάγνωση του γράφου από το αρχείο εισόδου και δημιουργούνται λίστες γειτνίασης. 
Οι λίστες είναι ταξινομημένες σε αύξουσα σειρά.

Για την εύρεση συνάντησης:
- Εκτελείται BFS λαμβάνοντας υπόψη την παρατικότητα (even/odd βήματα).
- Αν βρεθεί κόμβος όπου φτάνουν και οι δύο στον ίδιο χρόνο, γίνεται ανακατασκευή της διαδρομής.

Αν δεν υπάρχει συνάντηση:
- Στον μη κατευθυνόμενο γράφο εντοπίζεται ένα μονοπάτι και προστίθεται μία ακμή.
- Στον κατευθυνόμενο γράφο δοκιμάζονται ακμές που δημιουργούν κύκλους μήκους 2 ή 3.
- Για τον directed γράφο χρησιμοποιείται BFS σε καταστάσεις (Alice, Bob).

## Εκτέλεση

Για μη κατευθυνόμενο γράφο:
py rendezvous.py graph_1.txt

Για κατευθυνόμενο γράφο:
py rendezvous.py -d graph_1.txt

## Μορφή αρχείου εισόδου

n m
x1 y1
x2 y2
...
xm ym
A B

- n: αριθμός κόμβων
- m: αριθμός ακμών
- xi yi: ακμές
- A: αρχικός κόμβος Alice
- B: αρχικός κόμβος Bob
